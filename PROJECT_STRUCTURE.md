## Quick Commands

### Build
```bash
make              # Build both serial and OpenMP versions
make serial       # Build only serial
make openmp       # Build only OpenMP
make debug        # Build in debug mode
make clean        # Remove build files
```

### Run
```bash
cd examples/basic_cavity

# Serial
../../build/bin/ldc_serial

# OpenMP (set threads first)
export OMP_NUM_THREADS=4
../../build/bin/ldc_openmp
```

### Visualize
```bash
cd examples/basic_cavity
python ../../src/python/visualization/visualize_ldc.py
python ../../src/python/visualization/benchmark_ldc.py
```

## Build Options Reference

CMake options (use with `cmake .. -DOPTION=VALUE`):
- `CMAKE_BUILD_TYPE`: Release (default) or Debug
- `BUILD_OPENMP`: ON (default) or OFF
- `BUILD_SERIAL`: ON (default) or OFF
- `ENABLE_PROFILING`: OFF (default) or ON

Environment variables:
- `OMP_NUM_THREADS`: Number of OpenMP threads
- `CXX`: C++ compiler to use
